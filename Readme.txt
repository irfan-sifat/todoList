This is Irfan's ToDo App

Setup Guide:
	Step-1. Run your laravel backend with the command 'php artisan serve'
	Step-2. Create a database
	Step-3. Setup the .env file
	Step-4. Run the command 'php artisan migrate'
	Step-5. Run your vuejs fontend with the command 'npm run dev' or any other command
	Step-6. Collect your backend ip/domain and setup frontend API url
	Step-7. Goto axios.js in the frontend and adjust your API baseURL with backend

User Guide:
	Step-1. Please register if you don't have any account.
	Step-2. Login
	Step-3. Create, Read, Update and Delete your task
	Step-4. Enjoy!
